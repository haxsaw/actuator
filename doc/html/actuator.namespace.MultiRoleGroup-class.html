                                          MultiRoleGroup

Allow the creation of multiple RoleGroup based on a key.

This is a convenience class that simply wraps a RoleGroup in a MultiRole so that a group of roles can be created with each new key used to index into the group. Like MultiRole, this class behaves like a dict, where keys are immutable and coerced to strings.

See the base classes MultiRole and VariableContainer for more info on instantiation and usage.

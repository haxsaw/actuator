source code 

Take a collection of model reference objects and compute the Provisionables needed to satisfy the refs. An optional collection of model refs can be supplied and any ref in both collections will be skipped when computing the Provisionables.

Parameters:
  • modelrefs - an iterable of InfraModelReference instances for the model this SystemSpec instance is for
  • exclude_refs - an iterable of InfraModelReference instances whioh should not be considered when computing Provisionables (this will be skipped)

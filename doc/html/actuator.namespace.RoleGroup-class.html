Defines a group of Roles that are used together; a kind of ComponentGroup.

This class allows a set of Roles to be grouped together so that the group can be used easily in single model. The use cases for this tend to be situations where a set of Roles all are associated to a single IPAddressable in an infra model, as otherwise it gets a bit clumsy to name different IPAddressables to associate with each Role in the group.

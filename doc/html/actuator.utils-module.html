capture_mapping(domain, from_class)

source code 

Internal; a decorator that maps one class to another relative to a specific usage domain. Typically used to map a class to another, such that the second has some understanding of how to process the first. Often used to map data objects (and their derived classes) to handler classes.

find_file(filename, start_path=None)

source code 

Helpful utility that finds a file relative to some starting point.

Parameters:
  • filename - The name of the file to find.
  • start_path - A path prefix to use as the place to start looking for the file; if unspecified, will use the current value of os.getcwd() to determine the starting poing.
Returns:
the full path to the file, or None if it can't be found.

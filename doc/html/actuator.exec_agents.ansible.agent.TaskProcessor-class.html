_make_args(self, task)

source code 

Supplies the module-specific arguments for the processor. Must return a dict of Ansible Runner keyword args that are relevant to the module the class is for. The derived class must override this method.

result_check(self, task, result, logfile=None)

source code 

Checks the result of a Ansible Runner invocation. If there is a problem, raise ExecutionException with helpful data in the response.

Parameters:
  • task - the Task that was performed
  • result - The result dict returned from the Runner
  • logfile - If present, a file-like object that log messages will be written, regardless of the log level.
Raises:
  • ExecutionException - Raised if the result looks bad; the result itself will be formatted and added to the exception in the exception's 'response' attribute.
